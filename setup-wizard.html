ข้อมูลโรงเรียน

กรอกข้อมูลพื้นฐานของโรงเรียน ข้อมูลเหล่านี้จะแสดงบนเว็บไซต์ทั้งหมด

วาง URL รูปโลโก้จาก Google Drive หรือ Google Photos

เชื่อมต่อ Google Sheet

ข้อมูลทั้งหมดของเว็บไซต์จะเก็บใน Google Sheet ของคุณ ทำตามขั้นตอนด้านล่าง

ถ้ายังไม่มี Google Sheet ให้สร้างใหม่ก่อน โดยไปที่ sheets.google.com แล้วสร้าง Spreadsheet ใหม่
1
เปิด Google Sheet ของคุณ แล้วไปที่เมนู Extensions → Apps Script
2
ลบโค้ดเดิมออก แล้ว วางโค้ดด้านล่างนี้ ลงไปทั้งหมด
3
กด Save แล้วกด setupSheets เพื่อสร้าง Sheet อัตโนมัติ (Run ครั้งแรก)
4
กด Deploy → New deployment → Web app
ตั้งค่า: Execute as = Me, Who has access = Anyone
5
Copy URL ที่ได้ แล้วกรอกในช่องด้านล่าง
กำลังโหลดโค้ด...
URL จะขึ้นต้นด้วย https://script.google.com/macros/s/

ตั้งรหัสผ่าน Admin

ตั้งชื่อผู้ใช้และรหัสผ่านสำหรับเข้าสู่ระบบ Admin Panel

รหัสผ่านจะถูกเก็บไว้ใน Google Sheet ของคุณ สามารถเปลี่ยนได้ตลอดเวลาจาก Admin Panel
รหัสผ่านเริ่มต้นคือ admin / admin123 — กรุณาเปลี่ยนเพื่อความปลอดภัย!

🎉 ตั้งค่าเสร็จสมบูรณ์!
